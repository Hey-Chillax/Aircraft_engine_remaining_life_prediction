# 航空发动机剩余寿命预测 (Aircraft Engine RUL Prediction)

## 📋 项目简介

本项目是一个**课程设计项目**,基于NASA C-MAPSS涡扇发动机退化仿真数据(FD001子集),通过数据驱动的方法预测航空发动机的剩余使用寿命(Remaining Useful Life, RUL)。这是一个典型的预测性维护问题,对于提高航空安全性和降低维护成本具有重要意义。

### 项目背景
在智能制造领域,设备的意外故障可能导致生产中断、安全隐患和昂贵的维修成本。借助物联网技术和数据驱动的预测方法,我们能够实时监测设备健康状况,预测剩余使用寿命,从而为设备维护提供最佳决策支持。

## 🎯 项目目标

### 课程设计要求
1. **数据处理与可视化**
   - 对训练集和测试集进行预处理(标准化、特征工程)
   - 绘制时间序列趋势图、相关性热力图、分布图等
   - 使用pandas、matplotlib、seaborn等工具

2. **RUL预测建模**
   - 实现基础算法(线性回归、随机森林、神经网络等)
   - 使用RUL_FD001.csv进行监督学习
   - 分析预测误差(RMSE)并展示预测效果
   - 包括数据预处理、模型构建、交叉验证、模型对比

3. **总结与反思**
   - 撰写不少于300字的总结
   - 说明数据分析挑战、算法优缺点、改进方案

### 技术目标
- 根据发动机的历史运行数据和传感器读数,预测发动机还能运行多少个周期
- 开发高精度的RUL预测模型
- 为航空公司提供提前维护规划的决策支持

## 📊 数据集概览

### 数据规模
- **训练集**: 100台发动机,20,631条记录,26个特征
- **测试集**: 100台发动机,13,096条记录,26个特征
- **RUL标签**: 100个标签值

### 特征类型
- **标识特征**: 单元序号、时间
- **操作条件**: 飞行高度、马赫数、油门杆角度
- **传感器数据**: 21个传感器特征(温度、压力、转速等)

### 数据特点
- ✅ 无缺失值
- ✅ 时间序列数据
- ✅ 多变量传感器读数
- ⚠️ 包含7个常量特征需要处理
